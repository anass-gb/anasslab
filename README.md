🚀 Installation et exécution
1. Cloner le projet

git clone https://github.com/ton-username/portfolio.git
2. Aller dans le dossier
cd portfolio
3. Installer les dépendances
npm install
4. Lancer le projet
npm run dev
🌐 Build production
npm run build
